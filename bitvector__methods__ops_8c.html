Detailed Description

Bitwise operation methods for BitVector.

Implements Python bindings for all bitwise operators: AND, OR, XOR, their in‑place variants, bitwise NOT, and truth‑value testing. Operations are performed word‑wise using the native C backend for high throughput on large vectors.
