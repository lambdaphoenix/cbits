Detailed Description

Equality and hashing for PyBitVectorObject.

Implements __eq__, __ne__, and __hash__ using the C backend’s comparison routines and Python’s hashing utilities. Hash values are cached inside the BitVector object and invalidated whenever the underlying data mutates.
