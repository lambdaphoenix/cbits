Clear callback for the cbits module.

Traverse callback for the cbits module.

Integrates the module state into Python's cyclic garbage collector by visiting all Python objects stored inside the module state. This includes the BitVector type and the BitVector iterator type.

Parameters
moduleThe module whose state is being traversed.
visitGC visit function provided by the interpreter.
argExtra argument passed through by the GC.
Returns
Always 0 to indicate successful traversal.
