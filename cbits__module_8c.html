Detailed Description

Module initialization for the cbits._cbits extension.

Implements the initialization logic for the BitVector C extension. This includes creating and registering the module’s type objects, setting up the per‑module state, installing metadata, and defining the garbage‑collection callbacks used by CPython to manage module lifetime.
