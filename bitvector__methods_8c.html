Detailed Description

Aggregates all Python-visible BitVector methods into one table.

Collects all method groups (basic operations, copying, rank queries, etc.) and exposes them as a unified PyMethodDef array used by the BitVector type specification.

This file defines:

  • the full method table BitVector_methods
  • all associated docstrings for Python‑level methods
