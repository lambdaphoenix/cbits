Rank table construction and rank queries.

This module implements:

  • bv_build_rank

This module isolates the rank subsystem from the core BitVector logic and integrates with the popcount dispatch mechanism provided by compat.h.

void bv_build_rank ( BitVector *  bv)

Build or rebuild the rank tables for a BitVector.

This populates super_rank[] and block_rank[] to support O(1) rank queries. After this call, bv->rank_dirty is cleared.
