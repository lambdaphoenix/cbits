Detailed Description

Public C API for the BitVector data structure.

Declares the stable, external-facing API for BitVector:

  • construction and destruction (bv_new, bv_copy, bv_free)
  • single-bit operations (bv_get, bv_set, bv_clear, bv_flip)
  • range operations (bv_set_range, bv_clear_range, bv_flip_range)
  • rank queries (bv_build_rank, bv_rank)
  • comparison and subvector search (bv_equal, bv_contains_subvector)

This header exposes only the public interface. Internal helpers and inline fast-paths are intentionally separated into bitvector_internal.h.
