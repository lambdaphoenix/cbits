Detailed Description

Runtime dispatch for fastest popcount block implementation.

Contains fallback, AVX2, and AVX-512 versions of block-level popcount, plus runtime CPU feature detection to select the best implementation.

uint64_t cbits_popcount_block_fallback ( const uint64_t *  ptr)

Fallback popcount block implementation.

Processes in 64-bit chunks, summing up cbits_popcount64 for each of 8 words. Used when no vector instructions are available.

Parameters
ptrPointer to at least 8 uint64_t words.

Total popcount of the 8 words.
