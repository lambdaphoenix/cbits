Detailed Description

Forward declaration of the cbits module definition.

Provides external visibility of the module's PyModuleDef so that other compilation units-such as state helpers, type initializers, or method implementations-can reference the module definition without introducing circular dependencies.

This header does not define initialization logic; it only exposes the declaration of the module object implemented in cbits_module.c.
