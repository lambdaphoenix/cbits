Detailed Description

Definition of the PyBitVector Python type.

Declares the core Python wrapper type for native BitVector objects. This header provides:

  • PyBitVectorObject - the Python object structure
  • the type specification used to create the Python type
  • bitvector_wrap_new - a helper for constructing new wrapper objects

It is included by all BitVector method implementations and iterator components, forming the central definition of the Pythonâ€‘level BitVector API.

Wrap a native BitVector in a new PyBitVector Python object.

Allocates a new Python wrapper object and assigns ownership of the provided BitVector pointer to it.

Parameters
typeThe PyTypeObjectrepresenting the BitVector type.

Return values

Note
The caller transfers ownership of bv_data to the returned object.
