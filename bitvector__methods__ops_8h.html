Detailed Description

Bitwise operations for BitVector.

Declares the Python bindings for all bitwise operators supported by the BitVector type:

  • __and__, __or__, __xor__, __invert__
  • in‑place variants (__iand__, __ior__, __ixor__)
  • truth‑value testing (__bool__)
