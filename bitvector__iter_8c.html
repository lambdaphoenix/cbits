Detailed Description

Implementation of the BitVector iterator.

Defines the iterator returned by BitVector.__iter__(). Iteration proceeds bit‑by‑bit using a cached 64‑bit word and a shifting mask to minimize indexing overhead and reduce Python/C boundary crossings.
