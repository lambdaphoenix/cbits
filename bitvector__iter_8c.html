Detailed Description

Implementation of BitVector iterator.

Implements the iterator returned by BitVector.__iter__(). Iteration proceeds bit‑by‑bit using a cached 64‑bit word and a shifting mask to minimize indexing overhead and reduce Python/C boundary calls.
