Detailed Description

Slicing and indexed access implemantation for BitVector.

Implements __getitem__ and __setitem__ for both integer indices and slice objects. Slice extraction uses fast bit‑shifting for contiguous ranges and falls back to per‑bit copying for stepped slices. Slice assignment accepts any iterable of truthy values and writes them into the target range.
